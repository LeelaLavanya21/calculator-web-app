# Calculator Web App

A simple calculator built using HTML, CSS, and JavaScript.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open the file `calculator.html` or `index.html` in a web browser
3. Use the calculator to perform basic arithmetic operations
